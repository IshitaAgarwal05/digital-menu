# Glacier Ice Cream - Premium Digital Menu 🍨

A modern, high-performance digital menu built on the **MERN** stack (MongoDB, Express, React, Node.js). It features an elite, app-like user interface designed for seamless product discovery, cart management, and WhatsApp-integrated checkout.

## 🚀 Key Features
- **Elite UI/UX**: Premium glassmorphic cards, custom coral-violet gradients, and smooth Framer Motion layout animations.
- **Sticky Filtering Engine**: A fully responsive, gapless sticky header system. Includes Brand filters, Category sidebars, Flavor chips (🍫 Mango, 🍓 Strawberry), and Search/Sort controls that freeze intelligently as you scroll.
- **Advanced Cart System**: Swipe-to-dismiss animated cart drawer with dynamic "Free Topping" progress tracking and one-click WhatsApp order generation.
- **Surprise Me (🎲)**: A playful feature that randomly selects a product and opens its high-detail modal.
- **Dynamic Badges**: Real-time "NEW" tags based on `launchingyear` and precise `% OFF` discount badges generated by MongoDB `$expr` queries.
- **Automated DB Seeding**: Includes a `sync_excel.py` script to effortlessly parse, clean, and sync a client's Excel product list directly into the MongoDB cluster, complete with flavor and volume extraction.

## 🛠️ Technology Stack
- **Frontend**: React 19, Vite, Tailwind CSS (customized), Framer Motion.
- **Backend**: Node.js, Express, Mongoose.
- **Database**: MongoDB (Atlas cloud or local `mongodb://localhost:27017/digital_menu`).
- **Data Sync**: Python (`pandas`, `pymongo`, `Pillow` for image optimization).

## 💻 Running Locally

### 1. Backend API
```bash
cd backend
npm install
npm start 
```
*(Requires `MONGODB_URI` environment variable. Defaults to `mongodb://localhost:27017/digital_menu`)*

### 2. Frontend React App
```bash
cd frontend
npm install
npm run dev
```
*(Uses `import.meta.env.VITE_API_URL` to route requests to the backend)*

### 3. Syncing Excel Data (Optional)
If you have a `Digital_Menu.xlsx` file:
```bash
pip install pandas pymongo requests Pillow openpyxl
python sync_excel.py
```
*(Parses the Excel file, downloads/optimizes images to `.webp`, and seeds the MongoDB database)*
